Ответь по пунктам примерный план решения задания ниже:
Вам необходимо разработать небольшой сервис поиска по контактам на FastAPI, реализующий следующий функционал:

1. Принимать запросы пользователей, содержащие:
	1.1 Контент в виде текстового атрибута -- имя контакта search_query: str
	1.2 Метаданные -- самостоятоятельно сформировать и обосновать необходимый список.
2. По переданному в запросе имени необходимо найти из приложенного синтетического списка контактов (contacts.txt,
 положить локально в таблицу sqlite), который может содержать опечатки, один/несколько наиболее релевантных контактов (либо не вернуть ничего) в формате result: [{ID: NAME}, ...]
3. Логировать необходимую информацию работы приложения, а историю запросов класть в локальную таблицу sqlite
4. Продумать обработку исключений и валидацию запросов.

В качестве решения принимается git репозиторий приложения и локальная демонстрация работы (swagger) на разборе.
